Website Notes

From this website, it's possible to add, edit, and destory book and checkout records. Please read this guide carefully so you don't accidentally destroy important information. If you have questions, please ask a previous manager for help.

Site Permissions: What can I do?

…When not signed in?

Non-signed in users can view the homepage, the checkout procedures page, and use the search box to browse book records.

…When signed in as a borrower?

Borrowers can sign in to view their personal Profile page, which lists their name, and any books they have checked out. Borrowers cannot change their name, email, or password.

…When signed in as a manager?

Managers can create new user accounts and manager accounts, check books in and out, add new books to the catalog, delete book and user records, download all book records to CSV or Excel, and get lists of all user emails, and all emails of users with checked-out books.

Users

Adding Users

In order to checkout a book, a borrower must have a user account. User accounts must be created with both the user and a manager present (typically during book checkout.) Go to Profile -> Add New User. Ask the borrower to fill in their name, email, and password.

Keep in mind:

  • To distinguish users, make each user provide their full name.
  • Unlike many sites, users cannot change their password, name, or email once the account has been created.
  • You are signed in as a manager. Don't allow the user to do anything on the site beyond adding their personal info.

Deleting Users

All Users -> User -> 'Delete'

Keep in mind:

  • Keep at least one Manager account at all times!
  • If you delete a user with checked-out books, those books will be assumed lost; book records will be updated or deleted accordingly.
  • Each user Profile lists the date of creation; you can use this to find old accounts.

Emailing Users

Profile -> 'All User Emails' lists the emails of all users with accounts.
Profile -> 'Borrower Emails' list just those users with checked out books.

Both lists can be c/p into Outlook.

Check In and Check Out

Check Out Books

To checkout a book to Some Student:

  1. Find the book's listing, and click 'Checkout'. If there's no Checkout link, this means that there are no available copies of that book.
  2. From the checkout page, select 'Some Student' from the drop-down menu, and click 'Submit'.

Check In Books

To check in books from Some Student:

  1. Go to 'Some Students' Profile page: 'All Users' -> 'Some Student'.
  2. The Profile page will list Some Student's books. Click on 'Checkin' next to each book you wish to check in.

Book Records

Adding Books

There are two ways to add books to the catalog:

  1. Add new copies to an existing book record using the 'update copies' form on the book's Entry page.
  2. Add a new book record with the 'New Book' form.

Keep in mind:

  • When using 'update copies', the number you enter is the new total number of copies. If a book has 5 current copies, and you add 2 copies, you want to enter 7, not 2.
  • When adding a new book record, check that the book you are adding isn't already in the catalog!

Deleting Books

There are two ways to remove books from the catalog:

  1. Remove copies of a book using the 'update copies' form on the book's Entry page.
  2. Delete all copies of a book with 'Delete'.

Record Backup

Profile -> 'Download CSV Records'
Profile -> 'Download Excel Records'

These methods allow you to backup the current catalog of books (although not user or checkout records) to a CSV or Excel file. It's recommended that you backup records at the end of each day you add, modify, or delete book records.
